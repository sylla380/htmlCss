Aide mémoire des sélecteurs CSS déja vus

  • * sélectionne Tous les élément de la page html
  • .ma-jolie-nav sélectionne Tous les élément ayant le nom de la class "ma-jolie-nav"
  • #mon-unique-id sélectionne Tous les élément ayant le nom de "mon-unique-id"
  • div, article, aside sélectionne toutes les balises <div>, <article> ET <aside>
  • nav asélectionne les liens <nav>
  • nav a:hover sélectionne tous les liens de la nav et active une animation lors du survol de la souris

Propriétés CSS utilisées dans cette section dans cette page

Decrivez leur usage et leurs valeurs possibles

  1. background: couleur du fond de l'élément
  2. color : couleur de la police
  3. float: effet flottant
  4. font-family: police
  5. font-size: taille de la police
  6. hauteur de l'élément
  7. left: position à partir de la gauche de la feêtre
  8. line-height: hauteur de l'interligne
  9. margin: marge extérieures
  10. padding: marges intérieurs
  11. position: position de l'élément dans la page (on a utilisé que la valeur (fixed) pour l'instant)
  12. text-align: alignement du texte
  13. text-decoration: changement des valeurs héritées (ici le souligné qui est standard pour les liens sous tous les navigateurs est supprimé)
  14. top: position à partir du haut de la fenêtre
  15. transition: temps de transition lors d'une animation (ici sur les liens de la barre de menus)
  16. width: largeur de l'élément par rapport a la taille de la fenêtre si on indique une valeur relative (%, em, ...)
